Gesendete Daten

Reduzierte GET-Anfrage

LANaxy sendet keinen JSON-Body. Die frei editierbare Query wird an die Zieladresse angehängt; Platzhalterwerte werden automatisch URL-kodiert.

Beispieladresse 
Query:
status={status}&guardian={guardien}&message={text}&timestamp={date}&action=red_on

Ergebnis:
https://ziel.local/hook?status=warning&guardian=proxmox_api_guardian_1&message=Node+nicht+erreichbar&timestamp=2026-07-20T20%3A15%3A00&action=red_on
Verfügbare Platzhalter {status} {guardian} / {guardien} {message} / {text} {timestamp} / {date} {title} {kind} Feste Parameter wie action=red_on können beliebig ergänzt werden. Ein Bearer-Token wird ausschließlich im Authorization-Header übertragen.
