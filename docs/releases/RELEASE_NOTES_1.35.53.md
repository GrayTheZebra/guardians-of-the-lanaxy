# LANaxy 1.35.53

## Änderungen

- Manuelles Zusammenführen von Incidents vollständig entfernt.
- Auswahl und Schaltfläche aus der Incident-Detailansicht entfernt.
- Merge-Route und zugehörige Datenbanklogik entfernt.
- Statusfilter „Zusammengeführt“ entfernt.
- Bereits als `merged` gespeicherte Incidents werden beim Start wieder als beendet behandelt.
- Bestehende Incident-Daten, Timeline-Einträge und Notizen bleiben erhalten.
