# Guardians of the LANaxy 1.22.5

## Änderungen

- Typ-Badges wie GLOBE, THREAD, SERVER oder MQTT verwenden jetzt die Farbe ihrer Guardian-Kategorie.
- Das bisher überall verwendete blaue Badge wurde entfernt.
- Badge-Hintergrund, Text, Rahmen und Hover-Zustand sind dezent an die Bereichsfarbe angepasst.
- Die Darstellung gilt für die Guardian-Auswahl und das Launchpad.
