# Guardians of the LANaxy 1.17.1

- Robuster Kopieren-Button für MiniGuard-Installationsbefehle, auch bei HTTP ohne Secure-Context-Clipboard-API.
- Installationsbefehl funktioniert sowohl in einer Root-Shell ohne `sudo` als auch als normaler Benutzer mit `sudo`.
- Installer prüft explizit auf Root-Rechte und liefert eine verständliche Fehlermeldung.
- Deinstallationshinweise unterscheiden Root-Shell und normale Benutzerkonten.
