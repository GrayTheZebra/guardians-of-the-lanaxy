# LANaxy 1.35.47

## Historie

- Bearbeitungsnotizen werden auch dann in der Guardian-Historie angezeigt, wenn der zugehörige Incident bereits außerhalb des gewählten Zeitraums beendet wurde.
- Ein Incident wird für die Historie zusätzlich berücksichtigt, sobald innerhalb des gewählten Zeitraums eine Bearbeitungsnotiz vorhanden ist.
- Reine OK-Prüfmeldungen bleiben weiterhin ausgeblendet; dokumentierte Notizen gehen dabei jedoch nicht mehr verloren.
