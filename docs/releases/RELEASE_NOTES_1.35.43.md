# Guardians of the LANaxy 1.35.43

## Incidents nur noch automatisch beenden

- Die Aktion „Incident manuell beenden“ wurde vollständig aus der Detailansicht entfernt.
- Die Massenaktion „Abschließen“ wurde entfernt.
- Der zugehörige HTTP-Endpunkt wurde entfernt.
- Offene Incidents können weiterhin quittiert und dokumentiert werden.
- Ein Incident wird ausschließlich automatisch beendet, sobald der Guardian wieder einen gesunden Zustand meldet.
