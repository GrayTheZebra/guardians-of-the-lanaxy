# Guardians of the LANaxy 1.28.4

- Zentraler Kompatibilitätsabgleich zwischen LANaxy- und MiniGuard-Version.
- Kompatibilitätsrichtlinie berücksichtigt Mindestversion, empfohlene Version und Protokollversion.
- Systemweiter Warnhinweis bei fehlender, zu alter oder inkompatibler MiniGuard-Version.
- MiniGuard-Verwaltung zeigt pro Agent `Kompatibel`, `Update empfohlen` oder `Update erforderlich`.
- Direkter Sprung vom Warnhinweis zum jeweiligen Update-Befehl.
- Die zuvor veraltete sichtbare LANaxy-Versionsnummer im Template-Kontext verwendet nun immer `APP_VERSION`.
