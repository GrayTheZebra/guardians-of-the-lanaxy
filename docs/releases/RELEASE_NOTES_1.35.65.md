# LANaxy 1.35.65

- Beacon-Laufzeitfehler werden im Guardian-Abhängigkeitsdiagramm rot dargestellt und mit der konkreten Fehlermeldung gekennzeichnet.
- Die Guardian-Übersicht warnt deutlich, wenn über aktive Rules kein erreichbarer Beacon verfügbar ist.
- Deaktivierte, fehlerhafte, stummgeschaltete und fehlende Beacons werden bei der Erreichbarkeitsbewertung berücksichtigt.
