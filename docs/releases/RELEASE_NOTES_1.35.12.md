# Guardians of the LANaxy 1.35.12

- Export-Schaltfläche weist darauf hin, dass Tokens und Zugangsdaten nicht exportiert werden.
- Import erkennt redigierte und fehlende Pflicht-Secrets anhand des Guardian-Schemas.
- Betroffene Guardians werden deaktiviert importiert.
- Fehlende Zugangsdaten werden in Importvorschau, Guardian-Liste und Bearbeitungsformular markiert.
- Nach dem Import weist eine Warnung auf die notwendige Ergänzung und Aktivierung hin.
