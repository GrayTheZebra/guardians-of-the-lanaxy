# LANaxy 1.35.32

- Löschen eines Guardians entfernt nun immer dessen Laufzeitstatus.
- Neu angelegte Guardians mit derselben ID übernehmen dadurch keinen alten Status mehr.
- Nach ausdrücklicher Bestätigung werden zugehörige Incidents, Timeline-Einträge und Benachrichtigungsdaten gelöscht.
- Verknüpfungen als betroffenes Mitglied fremder Incidents werden sauber entfernt.
- Gespeicherte Smart-History des gelöschten Guardians wird bereinigt.
