# LANaxy 1.35.24

## PBS Guardian

- Datastores können direkt im Guardian-Formular über die PBS-API erkannt und ausgewählt werden.
- Backup-Gruppen werden einschließlich Datastore, Namespace, Backup-Typ und Backup-ID übernommen.
- Verify-, Prune-, Sync- und Garbage-Collection-Jobs werden erkannt; Jobtyp und Job-ID werden gemeinsam gesetzt.
- Konfigurierte PBS-Remotes können automatisch ausgewählt werden.
- Bestehende gespeicherte API-Secrets werden beim Bearbeiten weiterhin sicher wiederverwendet.
