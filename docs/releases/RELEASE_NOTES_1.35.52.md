# LANaxy 1.35.52

- Beacon-Nachrichten verwenden jetzt den frei vergebenen Guardian-Namen als Hauptbezeichnung.
- Die technische Guardian-ID wird separat ausgegeben.
- Statusmeldungen wurden auf `alt -> neu: Meldung` gekürzt.
- Die Ausfalldauer wird bei Wiederherstellungen als eigenes Feld ausgegeben.
- JSON-Webhooks erhalten zusätzlich `guardian_name` und `guardian_id`; `source` bleibt aus Kompatibilitätsgründen erhalten.
