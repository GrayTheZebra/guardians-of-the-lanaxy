# LANaxy 1.35.64

## Guardian-Abhängigkeiten

- Die Guardian-Detailansicht zeigt jetzt den vollständigen Benachrichtigungsweg Guardian → Rule → Beacon.
- Passende Rules werden anhand derselben Guardian- und Gruppenfilter ermittelt, die auch der Benachrichtigungsversand verwendet.
- Direkte Beacons und zeitversetzte Eskalations-Beacons werden getrennt dargestellt.
- Deaktivierte oder fehlende Rules/Beacons bleiben sichtbar und werden eindeutig markiert.
- Bei Guardians ohne passende Rule weist die Detailansicht auf die fehlende Benachrichtigungsroute hin.
