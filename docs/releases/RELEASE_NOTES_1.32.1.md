# Guardians of the LANaxy 1.32.1

- Die Massenbearbeitung bleibt verborgen, solange kein Guardian ausgewählt ist.
- Die Auswahlzeile zeigt kompakt die Anzahl markierter Guardians.
- Zusatzfelder erscheinen nur für die gewählte Massenaktion.
- Einstellungsfelder werden mehrspaltig nebeneinander dargestellt.
- „Alle sichtbaren auswählen“ berücksichtigt aktive Suche und Filter.
- Auswahlstatus und Teil-Auswahl werden korrekt synchronisiert.
