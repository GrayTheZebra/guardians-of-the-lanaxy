# Guardians of the LANaxy 1.35.0

## Installation und GitHub

- Neuer Ein-Befehl-Installer `bootstrap.sh`.
- Installation lädt das aktuelle stabile GitHub Release statt den Entwicklungsbranch.
- SHA-256-Prüfung vor jeder Installation und jedem Update.
- Debian- und Ubuntu-Prüfung mit verständlicher Fehlermeldung.
- Bestehende Installation wird erkannt und über `update.sh` aktualisiert.
- Neue GitHub Actions für Syntax-/Kompilierungstests.
- Automatische Release-Erstellung bei Tags wie `v1.35.0`.
- Release enthält stabile Dateinamen für Installer und Prüfsumme.
- README und kurze Installationsdokumentation aktualisiert.
