# Guardians of the LANaxy 1.23.6

- Einklappbare Debug-Ausgabe für rohe KI-Antwort, vorhandene Beacons und Beacon-Auflösung.
- Rule-Vorschau zeigt den tatsächlich zugeordneten Beacon.
- Bestehende Beacons werden robuster über Name, ID, normalisierte Schreibweise und Typ erkannt.
- Ein eindeutig vorhandener MQTT-Beacon wird bei einer ausdrücklichen MQTT-/Beacon-Anforderung verbindlich zugeordnet.
- Vorhandene Beacons werden in der Vorschau sichtbar aufgelistet.
