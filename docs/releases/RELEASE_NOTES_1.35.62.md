# LANaxy 1.35.62

- mDNS via Avahi integriert und unter System schaltbar.
- LANaxy veröffentlicht `_http._tcp` auf Port 8090 und ist über `<hostname>.local` erreichbar.
- Aktuelle Systemzeitzone und lokale Systemzeit werden unter System angezeigt.
- Systemzeitzone kann direkt über LANaxy geändert werden.
- Privilegierte Systemaktionen laufen über einen eng begrenzten Root-Helper.
