# LANaxy 1.35.26

## Guardian-Konfiguration sofort sichtbar

- Nach dem Speichern eines Guardians wird das zur alten Konfiguration gehörende Laufzeitergebnis sofort verworfen.
- Die Guardian-Karte zeigt unmittelbar den Status `pending` und den Hinweis, dass eine neue Prüfung aussteht.
- Alte Meldungen und technische Details der vorherigen Konfiguration werden nicht mehr bis zur nächsten Prüfung angezeigt.
- Das Verhalten gilt für alle Guardian-Typen und ebenso für konfigurationsändernde Massenaktionen.
- Historische Verfügbarkeitszähler bleiben erhalten.
