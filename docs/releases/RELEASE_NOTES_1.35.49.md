# LANaxy 1.35.49

- Historien-Incidents werden nicht mehr pauschal grün dargestellt, nur weil sie beendet sind.
- Grün wird nur verwendet, wenn die gespeicherte Incident-Meldung tatsächlich einen Übergang zu `ok` enthält.
- Beendete Warning- und Critical-Incidents behalten ihre ursprüngliche Schweregradfarbe.
- Die absteigende Sortierung mit dem neuesten Incident oben bleibt erhalten.
