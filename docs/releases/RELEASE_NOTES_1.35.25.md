# Guardians of the LANaxy 1.35.25

## Guardian-Formulare nach Prüfmodus aufgeräumt

- Beim Proxmox Backup Server Guardian wird nur noch der zum gewählten Prüfmodus passende Zielbereich angezeigt.
- PBS-Server blendet Datastore, Backup-Gruppe, Job und Remote vollständig aus.
- Datastore, Backup-Gruppe, PBS-Job und PBS-Remote besitzen jeweils einen eigenen sichtbaren Bereich.
- Der Backup-Gruppen-Modus zeigt den benötigten Datastore direkt zusammen mit den Backup-Feldern.
- Die kontextbezogene Erklärung und die Prüfziel-Zusammenfassung wechseln passend zum PBS-Prüfmodus.
- Die bereits vorhandene modusspezifische Anzeige des Proxmox VE Guardians wurde geprüft und beibehalten.
