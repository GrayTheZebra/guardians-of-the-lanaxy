# LANaxy 1.35.54

- Aktive Rules ohne aktiven Beacon werden in der Rule-Übersicht deutlich als Warnung markiert.
- Die Rule-Karte zeigt die Anzahl aktiver Beacons sowie die insgesamt ausgewählten Beacons.
- Bei deaktivierten oder fehlenden Beacons erscheint der Hinweis, dass keine Benachrichtigung möglich ist.
