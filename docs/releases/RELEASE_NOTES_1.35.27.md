# Guardians of the LANaxy 1.35.27

## PBS-Assistent

- „Updates-/Neustart-Guardian anlegen“ wählt bei Bedarf automatisch den ersten verfügbaren MiniGuard aus.
- Ohne verfügbaren MiniGuard wird die Option wieder abgewählt und verständlich erklärt.
- Die Vorschau zeigt den geplanten Updates-/Neustart-Guardian nun ausdrücklich an.
- Serverseitig wird eine fehlende MiniGuard-Auswahl abgefangen.
- Bereits vorhandene Updates-Guardians für denselben MiniGuard werden nicht mehr doppelt angelegt; mit „Vorhandene aktualisieren“ werden sie aktualisiert.
