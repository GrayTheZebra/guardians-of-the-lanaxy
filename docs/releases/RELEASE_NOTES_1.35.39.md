# LANaxy 1.35.39

## Historie zeigt Incident-Quittierungen

- Quittierte Incidents werden in der Historie des zugehörigen Guardians angezeigt.
- Der Antwortzeitgraph erhält einen sichtbaren Marker am Zeitpunkt der Quittierung.
- Eine neue Incident-Liste zeigt Beginn, Ende, Quittierungszeitpunkt, Benutzer und Notiz.
- Die Zusammenfassung enthält die Zahl quittierter Incidents im gewählten Zeitraum.
- Neue Quittierungen und aufgehobene Quittierungen werden zusätzlich in der Incident-Timeline protokolliert.
