# Guardians of the LANaxy 1.24.7

- Guardian-Intervalle werden automatisch lesbar formatiert.
- Sekunden bleiben bei kurzen Intervallen erhalten.
- Größere Werte werden in Minuten, Stunden oder Tage umgerechnet.
- Beispiele: `60` → `1 min`, `21600` → `6 h`, `90000` → `1 Tag 1 h`.
- Die interne Konfiguration arbeitet weiterhin unverändert in Sekunden.
