# LANaxy 1.35.38

- PBS-Backup-Guardians zeigen Grenzwert, aktuelles Backup-Alter und die Überschreitung direkt in der Statusmeldung.
- Bei einem unkritischen Backup wird zusätzlich die verbleibende Zeit bis zur Warning-Grenze angezeigt.
- Grenzwerte und Abweichungen werden auch in den technischen Prüfdaten gespeichert.
