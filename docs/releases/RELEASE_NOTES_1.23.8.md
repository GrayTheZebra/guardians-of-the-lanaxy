# Guardians of the LANaxy 1.23.8

- KI-Pläne akzeptieren jetzt auch Guardian-Anzeigenamen und bekannte Aliasnamen.
- Anzeigenamen werden vor der Schema-Prüfung auf interne Guardian-Module abgebildet.
- Dadurch werden zugehörige Konfigurationsfelder wieder gegen das richtige Schema geprüft.
- Manuelle Guardian-Tests schreiben ihr Ergebnis sofort in den Laufzeitstatus.
- Kartenstatus, Meldung, letzte Prüfung und Verfügbarkeit aktualisieren sich direkt nach „Jetzt prüfen“.
- Dezimal dargestellte Timeout- und Retry-Werte werden beim manuellen Test robust gelesen.
