# Guardians of the LANaxy 1.30.0

- Persistentes Hardwareinventar pro MiniGuard mit frei vergebbaren Geräte-Aliassen.
- Inventarvergleich erkennt neue, entfernte und geänderte USB-, PCI-, Disk-, ZFS-, serielle und Backup-Einträge.
- Automatischer MiniGuard Inventar Guardian meldet unbestätigte Änderungen über Incidents, Rules und Beacons.
- Inventaränderungen können gezielt bestätigt werden.
- Proxmox- und PBS-Assistent markieren bereits vorhandene Guardians und können bestehende Daten aktualisieren.
- Konfigurationshistorie besitzt einen Unified-Diff zur aktuellen Konfiguration.
- MiniGuard-Inventar bleibt nach dem Abruf in der Agentverwaltung sichtbar.
